# Smart Appointment System

## Overview
A full-stack smart appointment booking platform with separate panels for Admin, Service Providers, and Users. The system manages appointment scheduling, slot booking, approvals, reviews, complaints, and shift management.

## Features
- User authentication and profile management
- Separate Admin, Service Provider, and User dashboards
- Admin approval system for service providers
- Service and category management
- Slot-wise and date-wise appointment booking
- Appointment cancellation by both users and service providers
- Review and rating system
- Complaint management module
- Service provider shift management
- Appointment tracking and status updates

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Project Structure
client/ -> Frontend
server/ -> Backend
